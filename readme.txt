# 📞 Service Calls System — מערכת קריאות שירות לעסקים

---

## 🧩 תיאור כללי

מערכת קריאות שירות מודולרית, רספונסיבית ומוכחת בשטח, אשר פותחה לצורך ניהול פניות לקוחות בצורה חכמה ויעילה.  
✅ **המערכת נמצאת בשימוש פעיל בסוכנות ביטוח עם מעל ל־400 עובדים!**  
היא בנויה כך שתתאים לעסקים קטנים וגדולים כאחד, עם תשתית גמישה הניתנת להרחבה.

---

## ✨ פיצ'רים עיקריים

- 📄 שליפה ועדכון של קריאות שירות
- 🌀 מעקב סטטוס לפי שלבים: חדש ➝ בטיפול ➝ טופל
- 🕓 חישוב SLA (Response & Resolution Time)
- 📦 ארכוב קריאות במקום מחיקה סופית
- 🔐 ניהול משתמשים + הצפנת סיסמאות (bcrypt)
- ⚙️ ניהול קטגוריות ומחלקות
- 🔑 JWT Authentication (הכנה לאימות מתקדם)
- 🧾 דוקומנטציה מלאה + קבצי HTML לתיעוד API

---

## ⚙️ טכנולוגיות בשימוש

- **Backend:** Node.js (Express) + MongoDB (Mongoose)
- **Frontend:** Vite + TypeScript + Tailwind CSS
- **פיתוח:** Nodemon, dotenv
- **אבטחה:** bcrypt, JWT
