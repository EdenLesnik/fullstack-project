# 📞 Service Calls System — מערכת קריאות שירות לעסק

---

## 🧩 תיאור כללי

מערכת קריאות שירות מלאה, פותחה כדי לתת מענה פשוט, מהיר ומסודר לניהול פניות לקוחות בעסקים — במיוחד בסוכנויות ביטוח שבהן יש צורך במעקב שוטף אחרי קריאות שירות, SLA, ומעקב אחרי סטטוס טיפול.

המערכת תומכת ב:
- ניהול קריאות שירות.
- מעקב סטטוסים (חדש ➔ בטיפול ➔ טופל).
- SLA — הגדרת זמני תגובה וטיפול.
- ארכוב קריאות במקום מחיקה מיידית.
- ניהול משתמשים והרשאות.
- ניהול קטגוריות ומחלקות תמיכה.

---

## ✨ פיצ'רים עיקריים

- 📄 **שליפת קריאות לפי סטטוס** — פעיל / בארכיון.
- 🛠️ **עדכון סטטוס קריאה אוטומטי לפי מעגל טיפול**.
- 🧑‍💻 **מערכת הרשמה וניהול משתמשים** (כולל הצפנת סיסמאות).
- ⚙️ **ניהול מלא של קטגוריות ומחלקות תמיכה**.
- 🕓 **חישוב SLA אוטומטי** — זמני טיפול וזמן תגובה.
- 📦 **ארכוב קריאות במקום מחיקה מיידית** — שמירה על היסטוריה.
- 🔐 **JWT Authentication** (הכנה להוספת אימות מלא בהמשך).
